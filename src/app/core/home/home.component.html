Na versão 1.0 deste módulo você poderá visualizar os pedidos, assim como alterar a informação de status dos mesmos.

Por exemplo: um pedido acabou de ser gerado pelo site, este terá o status como "processando", a partir do momento em que o administrador visualiza este pedido e deseja realizar a entrega, o mesmo pode mudar o seu status para "confirmado" e posteriormente para "em entrega", para depois mudar para "entregue" ou "cancelado".

IMPORTANTE: O cliente receberá um email toda vez que o status do pedido for alterado!
