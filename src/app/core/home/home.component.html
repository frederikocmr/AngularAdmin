Na versão 1.0 deste módulo você poderá:

  • Visualizar e alterar os status dos pedidos;

  • Visualizar os usuários que estão cadastrados;

  • Cadastrar e excluir produtos;

  • Visualizar formulários de contatos de usuários.


ALGUMAS INSTRUÇÕES

Pedidos

Clique aqui para ir aos pedidos!

Os pedidos são feitos no site e após isto, o registro é gerado no sistema, e então é possível visualizá-los neste módulo.

Cada pedido possui os produtos requisitados, a quantidade de cada um juntamente com sua medida, o valor total, o método de pagamento e a opção escolhidas pelo cliente, o endereço de entrega, o nome do destinatário, o email do usuário que gerou o pedido, a data de registro e por fim o ID do pedido.

E o mais importante aqui é o STATUS do pedido, onde quando feito, este se inicia com status de "PROCESSANDO", para depois outros status serem definidos neste módulo.

Por exemplo: um pedido acabou de ser gerado pelo site, este terá o status como "processando", a partir do momento em que o administrador visualiza este pedido e deseja realizar a entrega, o mesmo pode mudar o seu status para "confirmado" e posteriormente para "em entrega", para depois mudar para "entregue" ou "cancelado".

IMPORTANTE: O cliente receberá um email toda vez que o status do pedido for alterado!

Usuários

Clique aqui para visualizar os usuários!

Neste módulo você poderá visualizar os usuários que estão cadastrados, juntamente com as informações fornecidas por ele na hora do cadastro, tais como nome, documento (CPF/CNPJ), endereços, pedidos feitos, email, data de nascimento e outros.

Para visualizar mais informações do usuário, basta clicar no registro e então uma janela com mais informações abrirá.



ALGUMAS INSTRUÇÕES

Produtos

Clique aqui para ir aos produtos!

Os produtos são listados e aqueles existentes podem ser excluídos ou novos podem ser adicionados através do formulário de cadastro.

Cada produto possui nome, descrição, categoria, preço, medida e url da imagem.

Para adicionar novos produtos, basta preencher o formulário corretamente e clicar no botão cadastrar. Após isto, uma mensagem de sucesso aparecerá e então o seu produto será listado na loja. E para excluir, basta clicar no botão que se encontra ao lado da listagem de cada.

A funcionalidade de cadastro de categorias ainda não foi implementada.

Para cadastrar o URL da imagem, é necessário informar um link de uma imagem que está hospedada na internet. A funcionalidade de fazer upload de imagens ainda não foi implementada.

Para visualizar mais informações do produto, basta clicar no registro e então uma janela com mais informações abrirá.



ALGUMAS INSTRUÇÕES

Contatos

Clique aqui para visualizar os contatos!

A listagem dos contatos se refere à pagina de Contato do Site, onde o usuário preenche um formulário e então após cadastrar, o registro é feito e um email é enviado para o administrador.

Para responder o contato, utilize o email da empresa e envie para o email que foi cadastrado no registro do formulário de contato.

Para visualizar mais informações do contato, como mensagem e dados do usuário, basta clicar no registro e então uma janela com mais informações abrirá.
